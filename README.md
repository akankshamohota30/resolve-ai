# Aza — AI Customer Service Agent for Amazon India

🔗 **Live Demo:** https://akankshamohota30-aza-resolve-ai.vercel.app/

## Problem
Amazon India users face an average 38-minute resolution time 
for routine issues. Analysis of customer reviews showed a 69% 
severe service failure rate — costing an estimated ₹43.2B 
annually in support costs and lost retention.

## Solution
Aza is a live AI agent that resolves routine Amazon India 
customer service requests through natural conversation — 
no human agents needed.

**Handles:**
- Missing packages
- Returns & refunds
- Order cancellations
- Order tracking

## Results
| Metric | Before | After |
|--------|--------|-------|
| Resolution time | 38 minutes | 2 minutes |
| Improvement | — | 95% reduction |
| Wallet credit re-spend | — | 73% within 72 hrs |

## How It Works
- User describes issue in natural language
- Aza maps to 1 of 4 JTBD-defined user scenarios
- Resolves instantly via conversation or escalates with context
- Amazon Pay wallet refund issued immediately where applicable
- Session-level fraud scoring (0–100) with live UI badge
